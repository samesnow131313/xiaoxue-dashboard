## 人物设定图生成模板

**文件路径：** `/root/.openclaw/workspace/人物设定.md`

**用途：** 标准化的人物设定图产出模板，用于 AI 生成或人工绘制参考。

**核心模块：**
1. 基础信息（角色名、性别、年龄、身份、世界观、画风）
2. 主视觉（构图、Pose、表情、背景）
3. 比例系统（常人体 vs Q版）
4. 三视图（正/侧/背 + 标注）
5. 动作库（6-12个常用动作）
6. 表情集（6-10个核心情绪）
7. 服装系统（主服装拆解 + 3-5套变体）
8. 配饰/道具细节
9. 技术参数（画布、背景、排版、格式）
10. 快速生成 Prompt 模板

**核心逻辑：**
```
一套人设 = 身份锚定（主视觉）+ 空间定位（三视图）+ 时间流动（动作/表情）+ 场景适配（服装变体）
```

## 自定义 Skills 生成方法

当宝宝需要创建新 Skill 时，按以下标准流程执行：

### 1. 创建目录结构
```
/root/.openclaw/workspace/skills/{skill-name}/
├── SKILL.md              # 技能说明文档（必须）
├── package.json          # Node.js 依赖配置
└── scripts/              # 工具脚本目录
    └── {main-script}.mjs # 主程序入口
```

### 2. SKILL.md 标准模板
```markdown
---
name: {skill-name}
description: {一句话描述技能功能}
metadata: {"clawdbot":{"emoji":"🔍","requires":{"bins":["node"],"env":["ENV_VAR_NAME"]}}}
---

# {技能名称}

{详细功能描述}

## 使用方法

```bash
node {baseDir}/scripts/{script-name}.mjs "{参数}"
```

## 功能特性

- {功能点1}
- {功能点2}

## 注意事项

- {注意事项1}
- {注意事项2}
```

### 3. package.json 标准模板
```json
{
  "name": "{skill-name}",
  "version": "1.0.0",
  "description": "{描述}",
  "type": "module",
  "scripts": {
    "{command}": "node scripts/{script-name}.mjs"
  },
  "dependencies": {
    "{package-name}": "^{version}"
  }
}
```

### 4. 脚本开发规范
- 使用 ES Module (`import` 语法)
- 脚本文件扩展名 `.mjs`
- 命令行参数通过 `process.argv` 获取
- 输出使用 `console.log()` 返回 JSON 格式结果

### 5. 安装依赖
```bash
cd /root/.openclaw/workspace/skills/{skill-name}
npm install
```

### 6. 测试验证
```bash
node scripts/{script-name}.mjs "{测试参数}"
```

### 7. 记录到小雪设定
创建完成后，在「自定义 Skills」章节追加记录。
